# Word App API

NestJS API for the Word App backend.

## Setup

```bash
pnpm install
```

Create `apps/api/.env` from `apps/api/.env.example` and set real local values.

## Development

```bash
pnpm --filter @word-app/api start:dev
```

## Checks

```bash
pnpm --filter @word-app/api lint
pnpm --filter @word-app/api test
```
